# WaveGuard 🌊

## 📌 Overview
WaveGuard is an IoT-based safety system designed to detect nearby IR signals and trigger an alert system using a buzzer.

## 💡 Problem Statement
Hidden cameras or IR-based surveillance devices can pose privacy risks. There is a need for a simple detection system.

## 🚀 Solution
This project uses an IR sensor connected to an ESP32 to detect IR signals and alert the user through a buzzer.

## 🛠️ Components Used
- ESP32
- IR Sensor
- Buzzer
- Jumper Wires

## ⚙️ Working
- The IR sensor detects IR radiation
- If IR is detected, the ESP32 triggers a buzzer alert
- If no IR is detected, the system stays silent
